# 📦 Sistema de Controle de Estoque (Java)

## 🎯 Descrição

Aplicação de terminal em Java para gerenciamento básico de produtos em estoque. Permite cadastrar e listar itens, com foco em consolidar fundamentos de lógica e organização de código.

---

## ⚙️ Funcionalidades

* Cadastrar produto (nome e quantidade)
* Listar produtos cadastrados
* Menu interativo no terminal

---

## 🧱 Estrutura do Projeto

```bash
sistemaEstoque/
│
├── src/
│   └── Main.java
│
├── README.md
├── requisitos.md
```

---

## 🚀 Como Executar

### Pré-requisitos

* Java (JDK 17 ou superior)

### Passos

1. Clonar o repositório:

```bash
git clone https://github.com/danilo062/sistema-estoque
```

2. Acessar a pasta:

```bash
cd cd sistema-estoque/src
```

3. Compilar:

```bash
javac Main.java
```

4. Executar:

```bash
java Main
```

---

## 📌 Regras de Negócio

* Nome do produto não pode ser vazio
* Quantidade não pode ser negativa
